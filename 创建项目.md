## loader

```js
loader,相当于转换器或加载器webpack默认仅支持js和json两种文件格式，通过loader去支持其他文件类型并且把它们转换为模块，添加到依赖中

loader原理:loader本身是一个函数，接受源文件作为参数，返回转换结果

常用loader:
 1. babel-loader 转换ES6、ES7、JSX等语法
 2. css-loader   支持.css文件的加载和解析
 3. less-loader 支持less文件转换成css
 4. ts-loader 将TS转换为js
 5. file-loader 进行图片、字体等打包
 6. thread-loader 多进程打包js和css      

 webpack loader 网址:https://webpack.docschina.org/loaders/

css-loader: https://webpack.docschina.org/loaders/css-loader/

在code/demo12基础上,修改code/demo13 使用css-loader style-loader处理css
处理CSS需要两个loader
      
      style-loader:将样式通过<style></style>标签插入到<head></head>之间
	css-loader :用于加载.css文件，并且转为commonjs模块对象，可以使用import进行导入
1.安装css-loader style-loader
   cnpm install style-loader css-loader –D
2.修改webpack配置文件,webpack.config.js
//处理顺序为css-loader style-loader
module: {
rules: [
{
test: /\.css$/,
use: ['style-loader', 'css-loader'],
},
]
},

    说明: 
（1）使用css-loader必须要配合使用style-loader：
（2）css-loader的作用是帮我们分析出各个css文件之间的关系，把各个css文件合并成一段css；
（3）style-loader的作用是将css-loader生成的css代码插入到页面的head部分,嵌入<style></style>标签
（4）多个loader配合使用时，处理顺序是：从下到上，从右到左 的顺序；
（5）二者组合在一起使你能够把样式表嵌入webpack打包后的js文件中。
```
